# MedrixFlow 项目上下文

## 项目信息
- **路径**: `REDACTED_PATH/Desktop/medrix-flow`
- **来源**: 基于 ByteDance DeerFlow 二次开发，已完成品牌定制
- **GitHub**: `https://github.com/Citrus-bit/medrix-flow`
- **本地访问**: `http://localhost:1000`

## 技术栈
- **后端**: Python 3.12, FastAPI, LangGraph, LangChain, uv
- **前端**: Next.js 16, React 19, TypeScript, TailwindCSS 4, Shadcn UI, pnpm
- **状态管理**: TanStack React Query（服务端状态）+ localStorage（本地设置）
- **部署**: Docker + nginx 反向代理

## 架构（4 个服务）
| 服务 | 端口 | 说明 |
|------|------|------|
| LangGraph | 2024 | Agent 编排引擎 (lead_agent) |
| Gateway API | 8001 | FastAPI 网关 (`backend/app/gateway/`) |
| Frontend | 3000 | Next.js (Turbopack) |
| Nginx | **1000** | 统一反向代理入口 |

## 启动
```bash
双击 start.command          # 一键启动所有服务 + 自动打开浏览器
make dev                    # 或手动启动
make dev-daemon             # 后台启动
make stop                   # 停止所有服务
```

## 关键配置文件
| 文件 | 用途 |
|------|------|
| `config.yaml` | 主配置：模型定义、工具、sandbox、skills 等（`$VAR` 引用 .env） |
| `.env` | 环境变量：`HUAWEI_MAAS_API_KEY`, `TAVILY_API_KEY`, `JINA_API_KEY` |
| `config.example.yaml` | 配置模板（包含所有可用字段示例） |
| `backend/langgraph.json` | graph 入口 `medrix_flow.agents:make_lead_agent` |
| `backend/pyproject.toml` | 包名 `medrix-flow` |

## 配置加载链路
```
.env (load_dotenv) → config.yaml (yaml.safe_load)
  → AppConfig.resolve_env_variables（递归替换 $VAR）
  → AppConfig 单例缓存（mtime 自动热重载）
  → Gateway API 通过 get_app_config() 读取
```
核心文件: `backend/packages/harness/medrix_flow/config/app_config.py`

## 前端架构要点

### 路由结构
```
/                           → 重定向 /workspace
/workspace                  → 重定向 /workspace/chats
/workspace/chats/[thread_id] → 聊天页面
/workspace/agents/...       → Agent 页面
```

### 设置系统
- **Settings Dialog**: `frontend/src/components/workspace/settings/settings-dialog.tsx`
- 左侧导航 + 右侧内容，标签页：配置 / 外观 / 通知 / 记忆 / 工具 / 技能 / 关于
- 每个标签页独立组件，使用 `SettingsSection` 包装器
- 本地设置: `core/settings/local.ts` → localStorage `medrix_flow.local-settings`

### API 调用模式
```typescript
// 获取数据
const { data, isLoading, error } = useQuery({ queryKey: [...], queryFn: () => fetch(...) });
// 修改数据
const { mutate } = useMutation({ mutationFn: ..., onSuccess: () => invalidateQueries(...) });
```
参考实现: `core/mcp/hooks.ts` (GET/PUT 完整 CRUD 模式)

### 国际化
- 类型定义: `core/i18n/locales/types.ts`
- 英文: `core/i18n/locales/en-US.ts`
- 中文: `core/i18n/locales/zh-CN.ts`
- Hook: `useI18n()` → `t.xxx.yyy`

### UI 组件库
- 基础组件: `components/ui/` (Button, Input, Select, Switch, Dialog, ScrollArea...)
- 复合组件: `components/workspace/` (Sidebar, InputBox, Settings...)
- AI 组件: `components/ai-elements/` (ModelSelector, Reasoning...)
- Toast: `sonner` 库 (`toast.success(...)`, `toast.error(...)`)

## 后端 Gateway API 端点

| 路由 | 方法 | 文件 |
|------|------|------|
| `/api/models` | GET | `routers/models.py` |
| `/api/mcp/config` | GET/PUT | `routers/mcp.py` |

| `/api/skills` | GET/PUT | `routers/skills.py` |
| `/api/setup/config` | GET | `routers/setup.py` ⭐ 新建 |
| `/api/setup/models` | PUT | `routers/setup.py` ⭐ 新建 |
| `/api/setup/test-model` | POST | `routers/setup.py` ⭐ 新建 |
| `/api/setup/test-tool-key` | POST | `routers/setup.py` ⭐ 新建 |
| `/health` | GET | `app.py` |

### 模型配置 Schema
`backend/packages/harness/medrix_flow/config/model_config.py`:
- `name`, `display_name`, `description`, `use`（提供商类路径）, `model`
- `api_key`, `base_url`, `max_tokens`, `temperature`
- `supports_thinking`, `supports_vision`, `supports_reasoning_effort`
- `ConfigDict(extra="allow")` — 允许提供商特有字段

### 工具 API Key 来源
| 工具 | Key 来源 | 代码 |
|------|---------|------|
| Tavily | `config.yaml` tools 段 `api_key: $TAVILY_API_KEY` → model_extra | `community/tavily/tools.py` |
| Jina | `os.getenv("JINA_API_KEY")` 直接读环境变量 | `community/jina_ai/jina_client.py` |

**消息渲染流水线:**
```
hooks.ts 创建乐观占位 → useStream 连接后端 → 真实消息到达时清除乐观消息
groupMessages 分组: ai+reasoning → "assistant:processing" | ai+content → "assistant"
Thinking 提取优先级: reasoning_content > content[0].thinking > <think>标签
```
