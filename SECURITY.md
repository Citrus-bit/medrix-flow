# Security Policy

## Supported Versions

As medrix-flow doesn't provide an offical release yet, please use the latest version for the security updates.
Current we have two branches to maintain: 
* main branch for medrix-flow 2.x
* main-1.x branch for medrix-flow 1.x 

## Reporting a Vulnerability

Please go to https://github.com/bytedance/medrix-flow/security to report the vulnerability you find.
